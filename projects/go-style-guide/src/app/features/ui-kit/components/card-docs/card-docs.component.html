With Horizontal Rules

Full Width

A horizontal rule that spans the entire width of the card can be implemented by adding the go-hr class on an hr tag within the card's go-card-content.

With Padding

Also adding the go-hr--padding modifier class will cause the width of the horizontal rule to respect the padding of the card containing it.
