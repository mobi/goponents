Usage

The <go-card> component should be the default container for all content.

Bindings

showHeader

showHeader by default is true, but if set to false it will hide the header of the card even if one is passed.

shadow

shadow by default is true. Set it to false to remove the box-shadow from the card, which is useful when embedding cards within other cards.

With Horizontal Rules

Full Width

A horizontal rule that spans the entire width of the card can be implemented by adding the go-hr class on an hr tag within the card's go-card-content.

With Padding

Also adding the go-hr--padding modifier class will cause the width of the horizontal rule to respect the padding of the card containing it.

Nested Cards

Use [shadow]="false" on a nested card to remove its box-shadow, avoiding the double-shadow effect when embedding cards within other cards.
