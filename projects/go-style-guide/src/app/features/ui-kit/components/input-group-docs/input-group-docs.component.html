Usage

The go-input-group component extends form controls by adding text, buttons, or selects on either side. Use goInputGroupPrepend to add content before the input and goInputGroupAppend to add content after. Labels should be placed on the go-input-group, not on the inner go-input. For button addons, place the directive directly on go-button ΓÇö do not wrap in a span.
