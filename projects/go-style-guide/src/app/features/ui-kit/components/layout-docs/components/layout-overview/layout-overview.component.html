Usage

The <go-layout> component should only be used once, at the root level component (typically the app component). This component is used to create an application layout that includes routing, navigation, and slots in the header for a logo, a search component, and actions.

Bindings

There are none! Instead, there are four areas used for content projection: go-layout-header and go-layout-nav and go-layout-footer. These should be applied to the <go-header> and <go-side-nav> and <go-footer> components respectively.

go-layout-main can be used to add content that will be shown on every page. A good example of this would be notifications that show up at the top of every page. The go-layout-main area will show up at the top of every page below the header and if a header bar is being used in conjunction, it will be beneath the header bar as well.
