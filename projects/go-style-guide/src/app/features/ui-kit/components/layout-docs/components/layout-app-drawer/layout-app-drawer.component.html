Usage

This page details how to use the navAppDrawer binding associated with the <go-side-nav> component. For any other information related to <go-side-nav>, please reference the Side Nav Documentation.

Interface

NavAppDrawer is the root interface that the navAppDrawer binding requires.

NavAppDrawer.currentAppIcon

Required. Icon for current application. This should be a material theme icon.

NavAppDrawer.currentAppDisplayName

Required. Name of application to display.

NavAppDrawer.appDrawerConfig

Optional. An array consisting of applications user has access to. Must follow the requirements of NavAppDrawerItem interface:

NavAppDrawerItem.displayName - name of application to display

NavAppDrawerItem.icon - icon for application (material theme icon)

NavAppDrawerItem.url - url for application, will open in a new tab
