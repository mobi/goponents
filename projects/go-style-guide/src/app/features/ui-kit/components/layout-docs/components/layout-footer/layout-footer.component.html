Usage

The <go-footer> component should only be used once, at the root level component. It should be placed inside a <go-layout> and be given the attribute of <go-layout-footer>. This component will be rendered at the end of the main content on ever page.

Bindings

There are none! This component uses content projection to just put the footer in the correct place.

Content Projection

The go-layout uses the go-layout-footer for content projection. This should be applied to a <go-footer> element to add content to the end of the page. The content will be centered on the page.
