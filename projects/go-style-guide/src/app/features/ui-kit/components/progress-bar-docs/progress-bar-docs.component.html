Usage

The <go-progress-bar> component should be used for displaying unspecified wait time or progress of an activity.

Bindings

mode

mode is the input data which decides what type of progress bar either determinate or indeterminate. By default, mode is set to determinate progress bar.

value

value indicator is a mandatory param for determinate progress bar.

rightLabel

rightLabel is a right side optional label.

leftLabel

leftLabel is a left side optional label.

Linear Determinate

Determinate progress bar display's progress of an activity. This should be used where the percentage of the activity completion is known.

With labels (example)

Linear Indeterminate

Indeterminate progress bar is used when the user is asked to wait while something finishes. This should be used where the wait time of an activity is unknown.
