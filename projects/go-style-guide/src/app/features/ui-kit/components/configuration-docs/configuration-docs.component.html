Branding

The GoConfigurationService can be used to brand a few select pieces of our design system. Currently the only things that can be branded are the active indicators on the sidenav, accordions, and the left portion of the header.

Update Branding Color

The setBrandColor(color: string) method can be used to update the branding color by passing in a full 6-digit hex code.
