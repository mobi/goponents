Usage

The Go Tab Component should be used when multiple sets of data need to be rendered, but vertical space is a concern.

The go-tab-group element must be accompanied by go-tab children.

Bindings

theme

theme will set the appearance of the tab. light is the default theme and should be used in most places. The only time the dark theme should be used is when the tab lives inside a UI element that has a dark themed background by default.

Lazy Loading

By default, components rendered inside of the tabs are eager loaded. This means they will be initialized when the page get's loaded. The tab content can be lazy loaded by adding a ng-template to the body of the content
