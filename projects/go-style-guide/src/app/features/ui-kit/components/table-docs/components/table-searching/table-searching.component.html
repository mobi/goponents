Implementing Table Searching

By default, searching is disabled. To enable searching, you may pass an GoTableSearchConfig class to the searchConfig property on the GoTableConfig object.

searchable

This property determines whether the table is searchable.

searchTerm

This is the term that is being searched for. This is useful when implementing server side integration, as it is emitted as part of the GoTableConfig object and can be used for server side filtering.
See Server Integration for more details.
