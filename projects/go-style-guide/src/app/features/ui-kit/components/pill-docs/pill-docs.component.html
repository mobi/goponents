Usage

The <go-pill> component should be used to to show terms that are being used to filter results from a table. These should populate the table following the use of an off-canvas with table-specific categories.

Bindings

removable

This binding determines whether or not to show the remove icon within the pill

  • true
  • false

removed

This is an event that is emitted from the pill when a user clicks on the remove icon.
